Deletes a given tuple (record) from a space (DB). Note that the key specified must belong to a primary index or any other unique index.

  • space string space name.
  • key mixed query key.

update (self, space, index, key, oplist)
Updates a given tuple (record) in a space (DB). Note that the key specified must belong to a primary index or any other unique index.

Parameters:

  • self table connection object.
  • space string space name.
  • index number or string index identifier.
  • key number or string query key.
  • oplist table update operator list.

Returns:

    table. Updated or upserted record.
upsert (self, space, key, oplist, new_tuple)
Upserts a given tuple (record) in a space (DB). Note that the key specified must belong to a primary index or any other unique index. Upsert means update if it exists, insert if not.

Parameters:

  • self table connection object.
  • space string space name.
  • key mixed query key.
  • oplist table upsert/update operator list. These values are used for the update.
  • new_tuple table tuple to be used as the record value when inserting.

Returns:

    table. Currently returns an empty table if successful.

eval (self, exp, result)
Performs an eval operation in the tarantool server. I.e., it evaluates the given Lua code and returns the result in a tuple.

Parameters:

  • self table connection object
  • exp string containing the Lua expression to be evaluated.
  • result table the tuple where the result will be returned.

Returns:

    table If the Lua code evaluation was successful.
